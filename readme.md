# Order Generator / Order Accumulator

Aplicação como avaliação técnica.

## Tecnologias
- Backend: .NET 8 (Minimal API)
- Frontend: React + Vite
- Testes: xUnit

### Como rodar o backend
```bash
cd backend
dotnet run
- http://localhost:5000

### Como rodar o frontend
```bash
cd frontend
npm install
npm run dev
- http://localhost:5173

### Como rodar o backend-test
```bash
cd backend-Tests
dotnet test
